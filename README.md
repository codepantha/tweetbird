## ğŸ’» Getting Started <a id="getting-started"></a>

> To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need npm which is included with Node.js.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/codepantha/tweetbird.git
```

### Install

Install this project with:

```sh
  cd tweetbird
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm run dev
```

### Run tests

To run tests, run the following command:

```sh
  npm run test
```

### Deployment

>You can deploy this project using: [Vercel](https://vercel.com)
